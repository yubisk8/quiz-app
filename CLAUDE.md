## プロジェクト概要

- **プロジェクト名**: quiz-app
- **内容**: 一般常識クイズアプリ。ユーザーに一般常識に関する設問を出題し、回答・採点・結果表示を行う Web アプリケーション。
- **技術スタック**: HTML / CSS / JavaScript（バニラ JS、フレームワーク・ビルドツールなし）

## 開発・実行方法

- ビルド工程はなし。`index.html` をブラウザで直接開けば動作します。
- ローカルサーバーで確認する場合（`fetch` で外部 JSON を読む等のため）:
  ```bash
  python3 -m http.server 8000
  # ブラウザで http://localhost:8000 を開く
  ```

## コーディング規約

- **言語**: HTML5 / CSS3 / ES6+（モダンなバニラ JavaScript）。
- **HTML**: セマンティックなタグを使用する。アクセシビリティ（`alt`、`label`、適切な見出し階層）に配慮する。
- **CSS**: クラス命名は一貫性を保つ（BEM など）。レイアウトは Flexbox / Grid を基本とし、レスポンシブ対応を意識する。
- **JavaScript**:
  - `var` ではなく `const` / `let` を使う。
  - DOM 操作とクイズのロジック（データ・採点）は関数単位で分離する。
  - 設問データはロジックから切り離し、データとして管理する。
  - 外部ライブラリやフレームワークは原則追加しない（バニラ構成を維持する）。

## 設問データの扱い

- 設問は「問題文・選択肢・正解」を持つデータ構造で管理する。
- 新しい設問を追加する際は、データ部分のみの変更で完結できるようにロジックと分離しておく。
